# AI & DATA DRIVEN HR 2026 — ระบบบันทึกความคืบหน้างาน

ระบบบันทึกและติดตามความคืบหน้าของแผนงาน **HR Transformation Roadmap (Phase 1: 5 Strategic Initiatives)**
เป็นเว็บแอปไฟล์เดียว (single-file) เปิดใช้งานได้ทันทีในเบราว์เซอร์ ไม่ต้องติดตั้งอะไร และบันทึกข้อมูลอัตโนมัติในเครื่อง (LocalStorage)

## ✨ ความสามารถ

- **Dashboard ภาพรวม** — วงแหวน % ความคืบหน้ารวม, นับงานตามสถานะ, แถบความคืบหน้าราย Initiative
- **5 Initiatives** ตามแผน: Leadership Coaching, AI & Data Skill, Smart Recruitment, People Analytics, Executive Dashboard
- **ขั้นตอนการดำเนินงาน (Step-by-step)** — แต่ละงานเพิ่มขั้นตอนย่อยได้ พร้อม **สถานะ** (วางแผน/กำลังทำ/เสร็จสิ้น/ล่าช้า) และ **วันที่คาดว่าจะเสร็จ**
- **คำนวณอัตโนมัติ** — % และสถานะของงานคิดจากขั้นตอนย่อย พร้อมธงเตือน ⏰ เลยกำหนด / ใกล้ครบกำหนด
- **สรุป & ไทม์ไลน์อัตโนมัติ** — สร้างข้อความสรุปความคืบหน้าและจัดกลุ่มงานตามช่วงเวลาให้เอง
- **ประวัติการอัปเดต** — บันทึกทุก action อัตโนมัติ พร้อมกิจกรรมล่าสุด
- **KPI Tracking** ต่อ Initiative
- **สองภาษา** ไทย/อังกฤษ · **โหมดมืด/สว่าง**
- **Export / Import (JSON)** และ **พิมพ์รายงาน**

## 🚀 วิธีใช้งาน

เปิดไฟล์ [`index.html`](index.html) ด้วยเบราว์เซอร์ (Chrome / Edge) ได้ทันที

## 🗂️ โครงสร้าง

```
index.html   # ทั้งระบบอยู่ในไฟล์เดียว (HTML + CSS + JavaScript)
```

## 💾 การเก็บข้อมูล

ข้อมูลบันทึกอัตโนมัติในเบราว์เซอร์ของแต่ละเครื่อง (LocalStorage)
แนะนำให้กด **ส่งออก (Export)** เก็บไฟล์สำรองเป็นระยะ และใช้ **นำเข้า (Import)** เมื่อย้ายเครื่อง
